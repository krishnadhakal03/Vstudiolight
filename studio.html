📖 Complete Usage Guide
🏠 Step 1 — Design locally (this app)
Open studio.html in any browser. No internet needed. Design your scenes, pick colours, write narration. Live preview updates as you type.
⚡ Step 2 — Generate Colab code
Click the ⚡ Generate All Colab Cells button. This creates 5 Colab cells tailored to your exact scenes and settings. Use the 📋 Copy buttons to copy each cell.
🔗 Step 3 — Open Google Colab
Go to colab.research.google.com. Create a new notebook. Make sure to enable GPU: Runtime → Change runtime type → T4 GPU.
📋 Step 4 — Paste cells in order
Create 5 code cells. Paste Cell 1 → Cell 2 → Cell 3 → Cell 4 → Cell 5 into each. Run them top to bottom with Shift+Enter. Cell 1 installs deps (~2 min). Cells 2–4 are instant. Cell 5 renders + downloads.
⬇ Step 5 — Download your MP4
Cell 5 auto-triggers a browser download of your finished video. It's H.264, AAC audio, 1080×1920 — ready to upload to YouTube Shorts, TikTok, or Reels.
🎛 Scene Layout Types
hook
Left-anchored bold stack with accent bar strip. Best for opening scene.
bigstat
Giant centred number/word with glow layers. Best for key data points.
split
Bold headline + dark pill body text box. Best for problem/reveal scenes.
code
Terminal-style code block. Best for showing prompts or steps.
cta
Centred with subtle bounce animation. Best for call-to-action endings.
✨ FX Types
particles — Floating accent dots, great for stats
scanlines — Subtle TV lines, cinematic feel
glitch — Digital glitch shifts, great for hooks
pulse — Expanding ring from centre, great for CTA
⚡ Render time estimates (Colab Free T4 GPU)
5 scenes × 3s = 15s video → ~2–4 min render
7 scenes × 4s = 28s video → ~4–7 min render
10 scenes × 3s = 30s video → ~5–8 min render
GPU renders ~80–120 fps. CPU fallback ~15–25 fps.
